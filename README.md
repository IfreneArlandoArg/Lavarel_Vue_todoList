### How to Install

Clone this repository, create the database *todo_list* and run the migrations:

```
php artisan migrate
```

Install dependencies from npm:

```
npm install
```

Run on the browser:
```
php artisan serve
```

Run with Hot Reload:

```
npm run hot
```
